Search for GroupServer site members

Introduction
============

This egg provides the page_ that allows the site administrator to search
the members of a GroupServer_ site.

Page
====

The page ``admin_search_people.html``, in the context of the site, allows
the site administrator to search for a site member. It is currently
restricted to site members for privacy reasons.
